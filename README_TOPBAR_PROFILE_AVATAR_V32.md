# TONI V32 – Obere Profilanzeige durch Profilbild ersetzen

Änderung:
- Die bisherige Anzeige aus Vorname/Nachname, Rolle und Klasse oben rechts wird durch ein rundes Profilbild ersetzt.
- Wenn kein Profilbild gespeichert ist, erscheinen Initialen.
- Klick auf das Profilbild öffnet das Profilfenster zum Ändern.
- Nach Änderung des Profilbilds wird das Bild oben direkt aktualisiert.
- Bei Abmeldung wird die Anzeige zurückgesetzt.
