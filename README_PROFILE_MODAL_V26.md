# TONI V26 – Profilfenster über eigenen Namen

Änderung:
- Durch Klick auf den eigenen Namen im Anmeldebereich öffnet sich ein Fenster.
- Das Fenster zeigt:
  - Vorname
  - Nachname
  - Klasse
- Das Fenster kann über ein kleines X geschlossen werden.
- Zusätzlich kann es mit der Escape-Taste geschlossen werden.

Einrichtung:
1. `public/index.html` ersetzen.
2. Neu veröffentlichen.
3. Browser hart neu laden.
