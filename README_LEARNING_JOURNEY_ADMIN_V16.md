# TONI V16 – Admin/Tutor Lernreisen verwalten

Neue Funktionen:
1. Admin/Tutor kann neue Lernreisen anlegen.
2. Admin/Tutor sieht seine zugeordneten bzw. angelegten Lernreisen.
3. Lernreisen können geöffnet, bearbeitet und gelöscht werden.

Einrichtung:
1. In Supabase `sql/learning_journey_admin_v16.sql` ausführen.
2. `public/index.html` ersetzen.
3. Neu veröffentlichen.
4. Als Admin oder Tutor anmelden.
5. Bereich „Lernreisen verwalten“ im Dashboard nutzen.

Hinweis:
- Admins sehen alle Lernreisen.
- Tutoren sehen ihre eigenen Lernreisen.
