# TONI Auth V15 – Schließen-Kreuz für Registrierung

Diese Version ergänzt im Fenster „Registrierung abschließen“ ein kleines X oben rechts.

Verhalten:
- Klick auf X schließt das Fenster.
- In derselben Browser-Session öffnet sich das Fenster nicht sofort automatisch erneut.
- Wenn eine neue Verifizierungs-Mail angefordert wird, darf das Fenster wieder geöffnet werden.

Einrichtung:
1. `public/index.html` ersetzen.
2. Neu veröffentlichen.
3. Browser hart neu laden.
