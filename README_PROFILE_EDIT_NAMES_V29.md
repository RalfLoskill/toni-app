# TONI V29 – Vorname und Nachname im Profilfenster bearbeiten

Neue Funktion:
- Im Fenster „Meine Daten“ gibt es neben Vorname und Nachname einen kleinen Bearbeiten-Button.
- Klick auf den Button öffnet eine kleine Bearbeitungsfläche.
- Vorname und Nachname können geändert und gespeichert werden.
- Die neuen Namen werden im Profil gespeichert.
- Der angezeigte Name im Anmeldebereich wird danach aktualisiert.

Einrichtung:
1. In Supabase `sql/profile_update_names_v29.sql` ausführen.
2. `public/index.html` ersetzen.
3. Neu veröffentlichen.
4. Browser hart neu laden.
