# TONI V17 – Besserer Editor für Lernreisen

Änderungen:
- Eingabefelder sind gleich breit und stehen sauber untereinander.
- Stationen werden nicht mehr in einem großen Textfeld eingegeben.
- Jede Station wird über einzelne Felder angelegt: Stationstitel, Untertitel, Stationsbeschreibung.
- Aufgaben werden ebenfalls über Felder hinzugefügt: Aufgabentyp, Aufgabentitel, Kurzbeschreibung, Pflichtaufgabe ja/nein.
- Stationen können hinzugefügt, bearbeitet, gelöscht und verschoben werden.

Einrichtung:
1. `public/index.html` ersetzen.
2. Neu veröffentlichen.
3. Browser hart neu laden.
