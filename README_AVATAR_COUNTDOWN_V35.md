# TONI V35 – Profilfoto mit Countdown aufnehmen

Neue Funktion:
- Klick auf „Foto aufnehmen“ startet einen Countdown direkt im Kamerabild.
- Es wird von 3 nach 2 nach 1 zurückgezählt.
- Danach wird das Foto automatisch aufgenommen.
- Es ist nur ein Klick auf „Foto aufnehmen“ notwendig.

1. `public/index.html` ersetzen.
